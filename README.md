# 드림핵 스타일 CTF 디스코드 봇

`/문제 생성` 으로 GUI 메뉴를 띄워 문제를 만들고, 정답(플래그)을 맞힌 사람만
해당 문제의 비공개 풀이 스레드에 입장할 수 있는 봇입니다.

## 명령어
- `/문제 생성` — 버튼 메뉴로 문제 생성
- `/문제 삭제` — 문제 선택 후 삭제 (출제자/관리자만)
- `/문제 스코어보드` — 정답자 랭킹·점수·푼 문제·주력 티어

## 동작 흐름

1. `/문제 생성` → 임시(나만 보임) 패널이 뜸. 버튼:
   - **📝 문제 이름** → 이름 입력 모달
   - **🏴 문제의 답** → 플래그 입력 모달
   - **🏅 티어** → 티어 입력 모달. `브론즈1` 처럼 입력하면 태그는 `브론즈`,
     글 제목은 `[브론즈1] 문제명` 으로 표시됨 (드림핵식 세부 티어 인식)
   - **✅ 제출** → 세 항목이 모두 채워지면 활성화
2. 제출하면:
   - **`🚩-문제-게시판` 포럼**(없으면 자동 생성)에 문제가 **포스트(게시글)** 로 쌓이고,
     티어는 포럼 **태그**로 구분됨
   - 동시에 숨김 `🔒-풀이방-보관소` 채널 안에 **정답자 전용 비공개 풀이방 스레드** 생성
     (출제자는 자동 입장)
   - 포럼 글에는 **`문제의 답` 버튼**이 달림
3. 다른 사람이 글의 **문제의 답** 버튼 → 플래그 입력 모달
   - 저장된 플래그와 일치하면 → 그 문제의 **비공개 풀이방에 입장 권한 부여**
   - 틀리면 → "플래그가 틀렸습니다" (나만 보이는 응답)

## 점수
티어 기본 점수(브론즈 10 / 실버 20 / 골드 30 / 플래티넘 40 / 다이아 50 …)에
세부 레벨을 더해 계산합니다. 스코어보드는 점수순 랭킹과 각자 푼 문제를 보여줍니다.

## 셋업

```bash
cd discord-ctf-bot
npm install
cp .env.example .env      # DISCORD_TOKEN, GUILD_IDS 채우기
npm run build && npm start
# 또는 개발 모드: npm run dev
```

### 봇 권한 (개발자 포털 / 서버 초대 시)
- `Manage Channels` (포럼/풀이방 채널·태그 생성)
- `Create Public Threads`, `Create Private Threads`, `Manage Threads`, `Send Messages`, `View Channels`

인텐트는 `Guilds` 만 필요합니다 (메시지 내용 인텐트 불필요).

### .env
- `DISCORD_TOKEN` — 봇 토큰
- `GUILD_IDS` — 명령어를 즉시 등록할 서버 ID(쉼표 구분). 비우면 전역 등록(최대 1시간 소요).

## 데이터
문제 정보(이름·플래그·티어·스레드ID·정답자)는 `data.json` 에 저장되어
봇을 재시작해도 유지됩니다.

# 드림핵 스타일 CTF 디스코드 봇

`/문제 생성` 으로 GUI 메뉴를 띄워 문제를 만들고, 정답(플래그)을 맞힌 사람만
해당 문제의 비공개 풀이 스레드에 입장할 수 있는 봇입니다.

## 동작 흐름

1. `/문제 생성` → 임시(나만 보임) 패널이 뜸. 버튼:
   - **📝 문제 이름** → 이름 입력 모달
   - **🏴 문제의 답** → 플래그 입력 모달
   - **🏅 티어** → 티어 입력 모달 (브론즈/실버/골드 …)
   - **✅ 제출** → 세 항목이 모두 채워지면 활성화
2. 제출하면:
   - `티어-문제` 채널(없으면 자동 생성, @everyone 에게 숨김)이 만들어지고
   - 그 안에 **비공개 스레드(포스트)** 로 문제가 쌓임 (출제자는 자동 입장)
   - 공개 `🚩-문제-알림` 채널에 **"새 문제 등록"** 메시지 + `문제의 답` 버튼 게시
3. 다른 사람이 알림의 **문제의 답** 버튼 → 플래그 입력 모달
   - 저장된 플래그와 일치하면 → 그 문제의 비공개 스레드에 **입장 권한 부여**
   - 틀리면 → "플래그가 틀렸습니다" (나만 보이는 응답)

## 셋업

```bash
cd discord-ctf-bot
npm install
cp .env.example .env      # DISCORD_TOKEN, GUILD_IDS 채우기
npm run build && npm start
# 또는 개발 모드: npm run dev
```

### 봇 권한 (개발자 포털 / 서버 초대 시)
- `Manage Channels` (티어/알림 채널 생성)
- `Create Private Threads`, `Manage Threads`, `Send Messages`, `View Channels`

인텐트는 `Guilds` 만 필요합니다 (메시지 내용 인텐트 불필요).

### .env
- `DISCORD_TOKEN` — 봇 토큰
- `GUILD_IDS` — 명령어를 즉시 등록할 서버 ID(쉼표 구분). 비우면 전역 등록(최대 1시간 소요).

## 데이터
문제 정보(이름·플래그·티어·스레드ID·정답자)는 `data.json` 에 저장되어
봇을 재시작해도 유지됩니다.
